About the Project

Lane X is a Wi-Fi controlled surveillance RC car designed to demonstrate real-time wireless control, embedded system integration and live visual monitoring. The vehicle can be controlled through a mobile application and a web interface.

The ESP32 works as the main controller and hosts the wireless/web control interface. It receives movement commands and controls the motors, lights and camera-based surveillance functions.

Key Features

Wi-Fi Control

Operates wirelessly using ESP32-based communication.

Camera Streaming

Supports visual monitoring through the camera module.

Web Server Interface

Allows browser-based control of movement and functions.

Mobile App Control

MIT App Inventor app provides movement and speed control.

Speed Control

Speed can be adjusted for controlled movement.

LED Control

Front and back lights can be controlled from the interface.

Software

  • Arduino IDE
  • ESP32 Board Package
  • MIT App Inventor
  • HTML, CSS and JavaScript

How It Works

The ESP32 creates a wireless control environment and hosts a web server. The mobile app or browser sends commands such as forward, backward, left, right and stop. The ESP32 processes these commands and drives the motors through the motor driver. The camera provides live visual feedback for inspection and surveillance.
